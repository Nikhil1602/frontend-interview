## 🏗️ HTML (Hyper Text Markup Language)

- `<html>`, `<head>`, `<body>`, `<title>`, `<meta>`, `<link />`, `<style>`, `<script>` are the basic boiler-plate tags.
- `<div>`, `<h1>...<h6>`, `<p>`, `<ul>`, `<li>`, `<ol>`, `<button>`, `<a>`, `<img/>`, `<br/>`, `<hr />`, `<table>`, `<form>` are the widely used tags.
- `<label>`, `<input />`, `<select>`, `<option>`, `<textarea>` are the important form elements.
- `<br />`, `<hr />`, `<link />`, `<img />`, `<input />`, etc. are self-closing tags that ends with `/>`.
- `<table>`, `<tr>`, `<th>`, `<td>`, are example of your table elements.
- `<article>`, `<aside>`, `<details>`, `<figcaption>`, `<figure>`, `<footer>`, `<header>`, `<main>`, `<mark>`, `<nav>`, `<section>`, `<summary>`, `<time>` are the [semantic](https://www.youtube.com/watch?v=bOUhq46fd5g) tags.

### 📝 Attributes in HTML

- HTML attributes provide additional information about HTML elements.
- All HTML elements can have attributes.
- Attributes usually come in name/value pairs like: `name="value"`.
- E.g. `<a href="#home">Home</a>`, `<img src="..." alt="..." />`, here `href`, `src`, and `alt` are examples of attributes.

## ⚡ JS (JavaScript)

If HTML builds the structure and CSS styles it, JavaScript adds behavior. It’s what makes things happen on a webpage.

JavaScript is one of the core technologies of the web:
```
HTML → structure
CSS → design
JavaScript → interactivity
```
